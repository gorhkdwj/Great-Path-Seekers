# decisionlogs · 결정 기록

방향을 바꾸거나 이후 작업에 영향을 주는 **중요한 결정만** 기록한다. 사소한 수정은 worklog에만 남긴다. (규칙: CLAUDE.md §11.2, §11.3)

## 구성
- **개인 결정** → `logs/decisionlogs/<이름>.md` : 각 팀원이 자신의 작업에서 내린 결정. ID `D-<코드>-NNN`.
- **팀 전체 결정** → `logs/decisionlogs/_team.md` : 기술 스택, 구현 범위, 데이터 처리, 보안, 배포·제출 방식 등 팀 전체에 영향을 주는 결정. ID `D-NNN`(코드 없음).

## PRD의 기존 팀 결정(D-001~D-013)
제품 방향 결정 **D-001~D-013**은 이미 `docs/prd/careerpath-ax-planning.md §14`에 기록되어 있다. 중복 관리를 피하기 위해 이관·복사하지 않고 그대로 두며, **신규 팀 결정은 `_team.md`에서 `D-014`부터** 이어서 번호를 붙인다.

## 기록 형식
```
### D-<코드>-00N · 결정 제목        (개인)   /   ### D-014 · 결정 제목  (팀 공용 _team.md)
**상황** / **검토한 선택지** / **결정** / **근거** / **영향** / **재검토 조건**
```
