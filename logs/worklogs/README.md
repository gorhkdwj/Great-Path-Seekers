# worklogs · 팀원별 작업 이력

각 팀원은 **자신의 파일**(`logs/worklogs/<이름>.md`)에만 작업 이력을 기록한다. 다른 팀원의 worklog 파일은 수정하지 않는다. 주요 사용자 요청이 끝날 때마다 최신 항목을 위에 추가한다. (규칙: CLAUDE.md §11.1)

## 파일 목록 & 멤버 코드
| 팀원 | 파일 | ID 접두 |
|------|------|---------|
| 서윤범(팀장) | `서윤범.md` | `W-SYB-NNN` |
| 김윤서 | `김윤서.md` | `W-KYS-NNN` |
| 김재천 | `김재천.md` | `W-KJC-NNN` |
| 김하연 | `김하연.md` | `W-KHY-NNN` |
| 박하린 | `박하린.md` | `W-PHR-NNN` |
| 함다연 | `함다연.md` | `W-HDY-NNN` |

## 기록 형식
```
### W-<코드>-00N · 작업 제목
**요청** / **수행 작업** / **변경 파일** / **검증**(미실행 시 사유) / **판단 근거** / **결과**(완료·남은 작업)
```

- ID는 파일 소유자의 코드를 접두해 팀 전체에서 고유하게 만든다(예: 김재천 → `W-KJC-003`).
- 개인 작업 중 방향을 바꾸는 결정은 `logs/decisionlogs/<이름>.md`에, 팀 전체 결정은 `logs/decisionlogs/_team.md`에 기록한다.
