# meetinglogs · 팀 회의록

팀 회의 내용을 회의 1건당 파일 1개로 기록한다. 회의에서 내린 **팀 전체 결정은 회의록에만 남기지 말고 `logs/decisionlogs/_team.md`에 D-ID로 기록**하고, 회의록에서는 해당 D-ID를 참조한다. (규칙: CLAUDE.md §11.6)

## 파일 이름 규칙
```
YYYY-MM-DD_N차.md      예: 2026-07-08_3차.md
```
